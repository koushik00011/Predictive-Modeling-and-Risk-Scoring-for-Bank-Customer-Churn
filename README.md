# 🏦 Bank Customer Churn Prediction System

An end-to-end Machine Learning project developed to analyze customer behavior and predict churn risk using classification algorithms, feature engineering, explainable AI techniques, and interactive deployment with Streamlit.

---

# 📌 Project Overview

Customer churn is one of the biggest challenges in the banking industry. Retaining existing customers is significantly more cost-effective than acquiring new ones.

This project focuses on:
- Identifying high-risk customers
- Understanding churn-driving factors
- Building predictive ML models
- Deploying a real-time churn prediction application

The system combines:
- SQL-based data handling
- Python data analytics
- Machine Learning
- Power BI reporting
- Streamlit deployment

# 📂 Project Workflow

## 1️⃣ Data Preparation
- Connected SQL dataset with Python
- Cleaned and validated customer records
- Removed unnecessary columns
- Handled categorical variables

---

## 2️⃣ Feature Engineering

Created business-driven engineered features such as:

- Balance-to-Salary Ratio
- Product Density Indicator
- Engagement Product Score
- Age-Tenure Interaction

These features improved model learning and customer behavior representation.

# 🤖 Machine Learning Models

The following classification models were trained and evaluated:

| Model | Purpose |
|---|---|

Models were evaluated using:

- Accuracy
- Precision

# 🏆 Best Model Performance

## ✅ Gradient Boosting Classifier

| Metric | Score |
|---|---|
| Accuracy | 0.87 |
| ROC-AUC | 0.87 |
| F1-Score | Strong balanced performance |

The Gradient Boosting model achieved the best balance between churn detection and prediction reliability.

---

# 📈 Explainable AI (SHAP)

Implemented SHAP (SHapley Additive exPlanations) to interpret model predictions and identify major churn-driving factors.

Key insights:
- Higher age increased churn probability
- Inactive customers showed higher churn tendency
- Product behavior strongly influenced churn risk
- Geography contributed to churn variation
